# 📇 Random User Generator

A React application that fetches random user profiles from the **RandomUser.me API** and displays them in a clean Bootstrap card. Each profile includes name, phone number, gender, location and profile picture. Users can generate a new random profile with a single button click.

---

## 🚀 Features

- Fetches random user data from API  
- Displays user details in a responsive Bootstrap card  
- Shows profile image, name, phone, gender and location  
- "Change User" button to load a new user  
- Built using React functional components and hooks  
- Fully responsive UI using Bootstrap  

---

## 🛠️ Tech Stack

- React  
- React Bootstrap  
- Bootstrap 5  
- RandomUser.me API
